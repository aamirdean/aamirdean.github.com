post @ 2016-11-10

HTML5初学者

Git 教程：

  1. cd ..(如果你进错硬盘，这个操作可以后退一步，重新进入。也可以多后退几次重新进入。)

  2. 首先进入硬盘， E： 。

  3. 进入需要操作的文件夹 ，cd空格文件夹在之前进入硬盘的相对路径回车结束。
    例如：cd myblog/blog/source 或者是直接把文件夹拖进去 回车也可以。

  4. 创建一个新的博客 hexo new “” (双引号里面写你创建新博客的名字，在window系统里面必须是双引号，在mac里面 可以是单引号)。默认为md格式。
    4.

博客主题的下载：

  1. 进入 hexo.io官网 。点击进入主题（Themes） 。
  2. 选择你需要下载的主题，在主题的名字下面右键点击在新标签中打开链接。
  3. 进入到一个github页面，右侧有一个绿色的按钮（clone or download）。
  4. 单击，会出现一个模态框 点击（download ZIP）。
  5. 把下载的文件解压到你创建的blog的子文件夹themes（主题）里面。
  6. 修改解压文件夹的名字。例如：hexo-theme-alpha-dust-master 。只保留
    文件的名称部分前后缀删除，修改完就是这样alphadust。
  7. 用编辑器 （阿童木、sublime都可以）打开blog里面的_config.yml文件，在最下面把theme属性修改成你准备应用的主题的名字。这里就是alphadust了。

  8. 你执行hexo server命令重新进入一次。打开终端生成的本地服务器进行预览。

  9. 新建一个仓库 仓库名必须是：github的用户名.github.io 。

  10. 点击右侧绿色的按钮（clone or download）获取新建仓库的连接。然后克隆新建的仓库到本地(放在那都行)。
    命令行：git clone https://github.com/Linu316602402/linu316602402.github.io.git
    (新建的仓库连接)；

  11. 查看状态 git status （会显示你修改过的文件）。
  12. 添加更改记录（相当于保存吧）git add . 。
  13. 再次查看状态 git status。
  14. 提交添加过的文件git commit –m “这(双引号)里面填写修改的内容的描述，也可以不写”。
  15. 如果终端提示是 Piease tell me who you are 。你需要先执行git config –global user.email “这(双引号)里面填写你的注册邮箱”。
  16. 如果终端提示Did you mean this config . 你需要先执行git config –global user.name“这(双引号)里面填写你的有户名”。
  17. 如果第15，16步没有提示 可以忽略15.16步的执行。
  18. 到这里就可以推送一下文件了。git pull origin master。
  19. 然后 git push origin master。
  20. Window 系统需要输入密码。就可以用了
